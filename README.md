## 💜Flutter 기차 예매 서비스
### 💜프로젝트 소개

---
### 💜 주요 기능
 ##### ✏️ 출발역, 도착역 선택
 #####  ✏️ 선택하면 좌석 홈화면에 표시
 ##### ✏️ 좌석 선택 누르면 좌석 페이지 이동
 ##### ✏️ 좌석 선택 시 선택 좌석 색상 변화
 ##### ✏️ 예매하기 클릭 시 확인창 출력
